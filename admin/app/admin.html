Welcome

About

Welcome to STEMHacks Admin Panel! Brandon Barker wrote STEMHacks Admin Panel on April 6, 2015 to make organizing a Hackathon a simpler process. A user system with multiple different applets allows for a diverse toolset that will reduce the amount of time spent doing mundane tasks and increase productivity. Furthermore, the simplistic UI allows for any organizer inexperienced or not to have a head start in running their hackathon.

How do I use this?

To begin using STEMHacks Admin Panel click on one of the menu options on the left-hand side. Each menu pane has an informational section when you click the question mark placed on the side of the title. If you are still confused by using the tool feel free to send an email to us and we will try to get back to you as soon as possible. To ensure easy of use, all pages are based on Googles Material design guidelines to make the most natively feeling interface as possible.
